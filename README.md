# musicdisp
Personal display for music score by midi pc messages

```
sudo apt-get install libasound2-dev libjack-dev python3 python3-pip python3-tk
pip install -r requirements.txt
```

# Setlist
* Touch
* Frontmann Eier
* Kleine Gedanken
* Pelirrojo
* Batalla en el cielo
* Agnus Deutschland
